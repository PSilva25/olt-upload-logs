Instale as dependências:

Backend (cd file-upload-backend):

npm install
├── cors
├── dotenv
├── express-fileupload
├── express
├── nodemon
├── pg
├── sequelize-cli
└── sequelize

Executar servidor:
├── cd src
└── node index.js



Frontend (file-upload-frontend):

npm install
├── antd
├── axios
├── moment
├── react-dom
├── react-scripts
└── react

Executar aplicação:
├── cd src
└── npm start
